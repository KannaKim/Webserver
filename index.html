Welcome to My C++ Web Server

This web server was built from scratch using C++ and implements the following features:

  • TCP socket communication
  • HTTP/1.1 protocol support
  • GET request handling
  • HTML file serving
  • jpg file serving
  • 404 error handling

How It Works

The server follows these steps:

  1. Initializes Winsock2 for network communication
  2. Creates a TCP socket and binds it to port 8080
  3. Listens for incoming connections
  4. Accepts client connections
  5. Processes HTTP requests
  6. Serves requested files
  7. Closes connections properly
